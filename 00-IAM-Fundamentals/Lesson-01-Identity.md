Why do organizations need an Identity Provider?
Traditionaly days we had lots of application and lots user with username and password, to manage all this we had admin to take manually these are the huge work performed by admin like creating user name, password, and so on. also they are lots of possibitites or lack of control over user termination to remove user access which leads unauthorized access for applicaiton.

Here Identity provider solves this problem ensure all identities store in single directly helps in authentication and authorization of all integrated application also helps in managing user lifecyle and restrict unauthorized access
